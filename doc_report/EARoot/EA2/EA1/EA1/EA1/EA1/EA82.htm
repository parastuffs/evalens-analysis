Gérer les groupes de questions permet au superviseur de définir et constituer les collections de questions, et de maintenir leur historique. Ce processus permet aussi de consulter et filtrer les groupes de questions, ainsi que d'y faire une recherche, et ce sur l'ensemble des informations décrivant les groupes.<br/>Le processus garantit la cohérence de chaque groupe de questions et respecte la règle rule-0006.<br/><br/>Chaque groupe de questions est décrite par les éléments suivants :<br/><ul>
<li>Intitulé (multilingue) : texte ou descriptif du groupe</li><li>Identifiant : permet d'identifier univoquement le groupe</li><li>Règle : procédure ou expression qui régit l'utilisation du groupe</li><li>Paramètres : décrit l'application de la règle spécifiée</li><li>Formules régissant la gestion du groupe : interprétation du résultat du groupe</li><li>Représentation visuelle : représentation utilisée pour visualiser le groupe sur les différents supports. Elle est formée sur base d'un texte et d'une typographie fournis par l'utilisateur ou d'une fonction programmée.</li><li>Domaine applicatif : spécifie la(les) application(s) dans la(les)quelle(s) le groupe peut être utilisé</li><li>Propriétés : informations variables spécifique à chaque groupe, dépendant du domaine applicatif.</li><li>Relations entre les groupes : dépendance, précédence, exclusion mutuelle, etc.</li><li>Statut : validé, rejeté, en attente, prototype, etc.</li><li>Période de validité : période durant laquelle le groupe est utilisable.</li><li>Type de composition : qualificatif déterminant le type d'union des éléments qui le composent (ex : indissociable, modifiable, à déterminer à la création)</li></ul>
<br/>Un groupe peut être vide ou associé à un ensemble de questions.<br/>Pour chaque question appartenant à cet ensemble, on peut définir des propriétés supplémentaires.<br/><br/>Pour un groupe donné, ce processus fournit aussi les informations suivantes :<br/><ul>
<li>Les groupes en relation</li><li>Les questions composant le groupe</li><li>Les questionnaires dont il fait partie</li></ul>
<br/>Pour faciliter la mise à jour, les opérations suivantes sont fournies :<br/><ul>
<li>Cloner un groupe existant</li><li>Définir un groupe comme modèle utilisable pour créer de nouveaux groupes</li><li>Périmer un groupe à la date du jour</li><li>Ordonnancer les questions</li></ul><p/>
