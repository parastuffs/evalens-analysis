Gérer la liste des qualificatifs consiste à maintenir à jour les différents discriminants utilisés dans l'application, et de maintenir leur historique. Ce processus permet aussi de les consulter et les filtrer, ainsi que d'y faire une recherche, et ce sur l'ensemble des informations décrivant celles-ci.<br/>Le processus garantit la cohérence de chaque qualificatif (rule-0007).<br/><br/>Chaque qualificatif est caractérisé par les éléments suivants :<br/><ul>
<li>intitulé (multilingue) : dénomination du qualificatif</li><li>description (multilingue)</li><li>identifiant : permet d'identifier univoquement le qualificatif</li><li>domaine applicatif : spécifie la(les) application(s) dans la(les)quelle(s) le qualificatif  peut être utilisé</li><li>période de validité : période durant laquelle le qualificatif est utilisable</li><li>propriétés : informations variables spécifique à chaque qualificatif, dépendant du domaine applicatif.</li><li>valeurs par défaut à utiliser lorsque le qualificatif est sélectionné</li><li>ordre d'apparition</li><li>qualificatif par défaut : qualificatif à sélectionner par défaut</li><li>représentation visuelle : représentation utilisée pour visualiser le qualificatif sur les différents supports.</li></ul>
<br/>Pour faciliter la mise à jour, les opérations suivantes sont fournies :<br/><ul>
<li>Cloner un qualificatif existant</li><li>Définir un qualificatif comme modèle utilisable pour créer de nouveaux qualificatif</li><li>Périmer un qualificatif à la date du jour</li></ul><p/>
