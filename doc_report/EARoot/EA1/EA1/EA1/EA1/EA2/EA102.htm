Gérer les publications permet au gestionnaire de déclarer les différent supports publicitaires électroniques au sein de l'application, afin de les mettre à disposition aux gestionnaires de communication.<br/><br/>Chaque support publicitaire est défini par les informations suivantes :<br/><ul>
<li>Type</li><li>Intitulé</li><li>Description</li><li>Propriété : taille, ...</li><li>Lien vers le fichier</li><li>Campagne</li><li>Statut : publié, prototype</li><li>Période d'utilisation</li></ul>
<br/>Les types des supports publicitaires sont :<br/><ul>
<li>Affiches imprimables</li><li>Bannières publicitaires (réseaux sociaux, sites web, afficheurs électroniques)</li><li>Textes de communication</li><li>Slogans</li></ul><p/>
