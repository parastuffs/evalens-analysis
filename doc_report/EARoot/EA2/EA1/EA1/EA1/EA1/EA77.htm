Le superviseur doit gérer les modèles des messages pouvant être envoyés à la communauté universitaire dans le cadre de l'évaluation des enseignements.<br/>Ces modèles sont ensuite récupérés par le gestionnaire de communication afin de diffuser les messages comme décrit dans « <a href="javascript:top.guidLink('{A7410F4C-1F6E-4dab-9317-D18EE321F1FC}')"><font color="#0000ff"><u>Envoyer les messages</u></font></a> ».<br/><br/>Les modèles sont sous l'entière responsabilité du superviseur et ce dernier peut les mettre à jour à tout moment.<br/><br/>Pour certains types de message, le superviseur peut faire appel au service de traduction.<br/><br/>Chaque modèle est la combinaison d'un texte et d'une liste de distribution.<br/><p/>
